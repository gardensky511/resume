## 自己PR

### 「ユーザーに優しく」を工夫・提案し、実装までします

画面の一番上にタブを使うページを改修してる時でした。選択中のタブは上段に青い線をつけて区別するデザインでしたが、スマホーのステータスバーも同じく青い色だったため、一見どのタブを見ているのか分かりづらい状況でした。私は「現在どこを見ているのか」を分かりやすくすることがユーザーに安定感を与えられると思い、ステータスバーの色を変更することを提案しました。他のメンバーたちも私の提案で初めてその部分に気づき、色変更を同意してくれました。その結果、私があげた案を私が直接実装し、本番まで反映することができました。このようにただ開発だけではなく、ユーザーに優しいサービスを作るために工夫・提案し、実現までできることが私の強みです。

## やりたいこと

> **面白いサービス、使いやすいサービス**を作っていくこと

- 面白いサービスとは？
  - 他サービスと比べても明確な差別点・強みがあるサービス
- 使いやすいサービスにしたい理由は？
  - 作ったサービスをユーザーに「心地よく」使ってほしい、そして「ずっと」使ってほしい
- なぜ「サービス」を作りたいのか？
  - デザインもプログラミングも何かを作るための手段であるので、その手段を身につけて素敵な結果を出したいと思ったため
