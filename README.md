## 自己PR

### サービスをよりよくするために工夫・提案し、実現します

画面の一番上にタブを使うページを改修してる時でした。選択中のタブは上段に青い線をつけて区別するデザインでしたが、スマホーのステータスバーも同じく青い色だったため、一見どのタブを見ているのか分かりづらい状況でした。私は「現在どこを見ているのか」を分かりやすくすることがユーザーに安定感を与えられると思い、ステータスバーの色を変更することを提案しました。ディレクターもその考えに同意し、本番に反映することができました。このようにただ開発だけではなく、サービスをよりよくするために工夫・提案し、実現まですることができます。
